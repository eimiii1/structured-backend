# 🚀 next-sprints

> Mini sprints in Next.js — backend experiments, Tailwind styling, and frontend fun.


## 📝 Overview
`next-sprints` is a collection of **small, focused Next.js projects** designed to explore:

- **Frontend development** with **React** and **Tailwind CSS**  
- **Backend experiments** using **Next.js API routes**  
- **Fullstack integration** and small web app concepts  

Think of this repo as a **sandbox for fast, iterative learning** — each project is a “sprint” that’s quick to build, test, and improve.


## 📂 Projects
Each folder contains a separate project. Example mini-sprints:

| Project             | Description                            |
| ------------------- | -------------------------------------- |
| `weather-dashboard` | Weather info fetched from a public API |

> More projects will be added as new sprints are completed!
